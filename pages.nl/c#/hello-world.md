# hello-world

> Basissyntax voor het tonen van tekst naar de standaarduitvoer.
> Meer informatie: <https://learn.microsoft.com/en-us/dotnet/csharp/tour-of-csharp/tutorials/hello-world>.

- Toon een eenvoudige tekenreeks:

```csharp
using System;

namespace HelloWorldApp {
    class Hello {
        static void Main(string[] args) {
            Console.WriteLine("Hello World");
        }
    }
}
```
